# Exchange Rate Telegram Bot

This bot accepts currency pair and returns it exchange rate.

Accepted formats:

1. USDMYR
2. USD MYR
3. USD-MYR

same, but lowercase.

Response:

1 USD = 4.1325 MYR

The bot can work as a calculator. For instance,

Request: `100 usdmyr`
Response: `100 USD = 413.25 MYR`
